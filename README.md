Viraj Hotel & Restaurant Website

This is a responsive and modern website developed for Viraj Hotel & Restaurant, located at Kalariya, Dhar Road, Indore.

The website is designed to provide customers with complete information about the restaurant, including menu, gallery, location, and table booking options. It is optimized for both desktop and mobile devices and follows a clean, user-friendly layout.
